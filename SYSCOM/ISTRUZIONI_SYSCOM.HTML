Libretto d'istruzioni per SYSCOM

Segui le seguenti istruzioni per utilizzare SYSCOM sul tuo computer.

1. Installa Powershell 7.4.2 o versioni successive

  • Assicurati di avere Powershell versione 7.4.2 o successiva installato sul tuo computer.
  • Se non l'hai già fatto, puoi scaricare Powershell dal seguente link: Scarica Powershell.
  • Dopo aver installato Powershell, puoi verificarne la versione digitando "pwsh.exe" nella barra di ricerca e premendo Invio.

2. Installa SYSCOM

  • Scarica il file di installazione INSTALL.PS1 dal seguente link: Pacchetto SYSCOM.
  • Salva il file in una cartella sul tuo computer.
  • Esegui il file di installazione con un doppio click o con il tasto Invio.
  • Se l'installazione è avvenuta correttamente, verrà creata una scorciatoia al file SYSCOM sul desktop.
  • A questo punto basterà fare doppio click o battere Invio sulla scorciatoia per avviare SYSCOM

OPPURE

3. Esegui il file SYSCOM.PS1

  1. Naviga nella cartella SYSCOM locale.
  2. Fai click sul pannello del percorso.
  3. Apri Powershell battendo nella barra del percorso "pwsh" e premi Invio.
  4. Verrà aperta una finestra del terminale di Powershell.
  5. Batti il comando .\SYSCOM.PS1 e premi Invio.
  6. CONSENTI AL PROGRAMMA DI APPORTARE MODIFICHE AL DISPOSITIVO
  7. Se hai eseguito correttamente le operazioni avrai questa schermata.

OPPURE

4. Imposta Powershell 7.4.2 o versioni successive come app predefinita per i file .PS1

  1. Fai clic con il tasto destro sul file SYSCOM.PS1.
  2. Seleziona "Apri con" e poi "Scegli un'altra app".
  3. Seleziona Powershell 7.4.2 o una versione successiva dalla lista delle app suggerite o clicca su "Altre app" per trovarlo.
  4. Assicurati di spuntare "Usa sempre questa app per aprire i file .PS1".
  5. Fai clic su "OK".

5. Apri il file SYSCOM.PS1 con un doppio clic

  • Una volta impostato Powershell come app predefinita per i file .PS1, sarà sufficiente fare doppio clic sul file SYSCOM.PS1 per aprirlo.
