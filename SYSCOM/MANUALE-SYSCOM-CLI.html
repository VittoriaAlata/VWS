Libretto d'istruzioni per SYSCOM

Segui le seguenti istruzioni per utilizzare SYSCOM sul tuo computer.

REQUISITO: Installa Powershell 7.4.2 o versioni successive

  • Assicurati di avere Powershell versione 7.4.2 o successiva installato sul tuo computer.
  • Se non l'hai già fatto, puoi scaricare Powershell dal seguente link: Scarica Powershell.
  • Assicurati di installare Powershell nella cartella predefinita
  • Dopo aver installato Powershell, puoi verificarne la versione digitando "pwsh.exe" nella barra di ricerca e premendo Invio.

Come avviare SYSCOM sul tuo computer

Passo 1: Installazione di SYSCOM

Fai doppio click sul file SYSCOM.msi per iniziare l'installazione.

Passo 2: Avvio di SYSCOM

Dopo l'installazione, fai doppio click sul collegamento di SYSCOM sul desktop per avviare il programma.

Passo 3: Autorizzazione dell'esecuzione dello script

Quando richiesto, autorizza l'esecuzione dello script con la firma specifica.

Passo 4: Conferma delle modifiche

Se viene richiesto di apportare modifiche al tuo sistema, clicca Sì per confermare.

Passo 5: Avvia SYSCOM

Se viene richiesto di apportare modifiche al tuo sistema, clicca Sì per confermare.
